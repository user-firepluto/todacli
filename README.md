# TodaMaster
O TodaMaster é um aplicativo simples que resume uma página do site "todamateria.com.br" ou "infoescola.com" com o Gemini 2.5 Flash e transforma em um teste de digitação
# Dependências
Para CLI: python(3.12 e superior), time, requests, os, textwrap, bs4, google
Para GUI: python(3.12 e superior), time, requests, webbrowser, bs4, google, PySide6
# Créditos
Gemini(Google) pela maior parte do código e <a href="https://www.flaticon.com/free-icons/electric-keyboard" title="electric keyboard icons">Electric keyboard icons created by yoyonpujiono - Flaticon</a> pelo ícone
# Nota
As dependências podem ser instaladas com o pip ou automáticamente com o pluto
